Status Glossary

Every status a song or album file can have — what it means and where it sits in the pipeline.

Song Statuses

Normal Flow

Status Meaning
{{ status_badge('new') }} Song added to the database, not yet searched on Soulseek.
{{ status_badge('searching') }} Currently searching for this song on Soulseek.
{{ status_badge('queued') }} Download enqueued with a Soulseek peer, waiting for transfer.
{{ status_badge('downloaded') }} File successfully downloaded to staging area.
{{ status_badge('delivered') }} File moved to destination folder — fully delivered.

Quality Upgrades

Status Meaning
{{ status_badge('upgrade_queued') }} FLAC download enqueued, replacing an existing MP3.
{{ status_badge('upgraded') }} FLAC replacement found and delivered for a previously MP3-downloaded song.

Issues

Status Meaning
{{ status_badge('not_found') }} No results found on Soulseek after searching all query variants.
{{ status_badge('no_match') }} Results found but none passed quality or filename filters.
{{ status_badge('stalled') }} Download stalled (no progress for 24 hours), alternative source queued.
{{ status_badge('stalled_waiting') }} Download stalled and maximum alternative sources (3) exhausted.
Missing file Song marked as downloaded in the database but the file is missing from disk. Shown on the Downloads page under "Needs Attention".

Albums

Status Meaning
{{ status_badge('downloading') }} Album folder download in progress — individual files have been enqueued, waiting for all to complete before marking as downloaded.

Other

Status Meaning
{{ status_badge('removed') }} Song removed from the source playlist or file — kept in the database for history.

Album File Statuses

{{ album_file_badge('downloaded') }} Individual album file successfully downloaded.
{{ album_file_badge('failed') }} Individual album file download failed.
